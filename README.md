## Getting Started

### Local Development
1. Clone or download the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required!

### Deployment Options
This website can be deployed to any static hosting service:
- **GitHub Pages** - Free hosting with GitHub repository
- **Netlify** - Free with automatic deployments from GitHub
- **Vercel** - Optimized for modern web
- **AWS S3** - Scalable cloud hosting
- **Traditional Web Hosting** - Any standard hosting provider

## Enable GitHub Pages

1. Go to your repository settings
2. Navigate to **Pages** section
3. Select **main** branch as source
4. Your site will be live at: `https://tovikasgarg.github.io/TIRUPATICONSULTINGGROUP/`

## Customization Guide

### Update Contact Information
Edit the contact section in `index.html`:
```html
<div class="info-item">
    <i class="fas fa-envelope"></i>
    <div>
        <h3>Email</h3>
        <p><a href="mailto:vikas@tirupaticonsulting.in">vikas@tirupaticonsulting.in</a></p>
    </div>
</div>
```

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e40af;
    --secondary-color: #0f766e;
    --accent-color: #f97316;
    /* ... other colors ... */
}
```

### Add Company Logo
Replace the text "TCG" logo in `index.html`:
```html
<div class="logo">
    <img src="path/to/logo.png" alt="TCG Logo" style="height: 40px;">
</div>
```
